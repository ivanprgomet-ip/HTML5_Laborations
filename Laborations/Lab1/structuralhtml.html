Explanations

Header
Used to contain the header content of a site. Normaly contains navigation for the website.
Footer
Contains the footer content of a site. Normally contains adresses, contact person, website, copyright etc.
Nav
Contains the navigation menu, or other navigation functionality for the page. Helps a user navigate around on the website with for example a navigational menu
Article
Contains a standalone piece of content that would make sense if syndicated as an RSS item, for example a news item.
Section
Used to either group different articles into different purposes or subjects, or to define the different sections of a single article.
Aside
Defines a block of content that is related to the main content around it, but not central to the flow of it.
