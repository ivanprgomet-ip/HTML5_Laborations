Structural elements html5 defined

<Header>
Used to contain the header content of a site. Normaly contains navigation for the website.
<Footer>
Contains the footer content of a site. Normally contains adresses, contact person, website, copyright etc.
<Nav>
Contains the navigation menu, or other navigation functionality for the page. Helps a user navigate around on the website with for example a navigational menu
<Article>
Contains a standalone piece of content that would make sense if syndicated as an RSS item, for example a news item.
<Section>
Used to either group different articles into different purposes or subjects, or to define the different sections of a single article.
<Aside>
Defines a block of content that is related to the main content around it, but not central to the flow of it.

Discover the tag world

<abbr> & <blockquote>
  • <abbr>: If you use an abbreviation or an acronym, then the abbr element can be used. An abbreviation is a shortened form of a word. ex: lb., amt., mgmt. A title attribute on the opening tag is used to specify the full term.
  • <blockquote>: The blockquote element is used for longer quotes that take up an entire paragraph. In short it indicates that a block of text is a quotation.
<acronym>
The acronym tag defines an acronym. An acronym is an initialism (initialism: An abbreviation formed from the initial letters of a phrase ex: HTML, SVG) that is pronounced as a word. ex: radar, scuba, JIT, WYSIWYG. An acronym must spell out another word. For example: NASA, ASAP, GUI. For example: can i get this asap?
<figure> & <figcaption>
  • Figure:The <figure> element is intended to be used in conjunction with the <figcaption> element to mark up diagrams, illustrations, photos, and code examples (among other things). The figure element represents a unit of content, optionally with a caption, that is self-contained, that is typically referenced as a single unit from the main flow of the document, and that can be moved away from the main flow of the document without affecting the document’s meaning.
  • Figcaption: The <figcaption> element represents a caption or legend for a figure. The <figcaption> element is optional and can appear before or after the content within the <figure>.

<table> & <col> & <colgroup> & <tr> & <th>
  • <table>: the table element is used to create a table. The contents of the table are written out row by row.
  • <tr>: You indicate the start of each row using the opening tr tag. (The tr stands for table row.)
  • <th>: The th element is used just like the td element but its purpose is to represent the heading for either a column or a row. (The th stands for table heading.)
  • <colgroup>: The colgroup tag specifies a group of one or more columns in a table for formatting.
  • <col>:To define different properties to a column within a colgroup, use the col tag within the colgroup tag.
